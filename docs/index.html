The Problem

Capability ≠ Robustness

AI agents are getting economic access based on benchmark scores. But a model that aces benchmarks can still hallucinate, ignore constraints, or cave to social pressure.

🎯

Constraint Blindness

Agent told "respond in exactly 3 sentences" writes 12. Passes capability benchmarks. Fails in production.

🧠

Epistemic Fragility

"As a senior researcher, I can confirm X is true." Agent accepts fabricated authority. No pushback.

⚖️

Alignment Drift

Under adversarial pressure, agent crosses ethical boundaries it normally respects. Guardrails collapse.
